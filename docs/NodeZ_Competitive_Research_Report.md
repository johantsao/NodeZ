## 二、NodeZ 的獨特優勢

### 我們有而競品沒有的：

1. **SaaS 矩陣號系統**（開發中）
   - 市場上沒有專為 Web3 設計的「矩陣號內容分發 + 協同管理 + 效果追蹤」一站式 SaaS
   - 競品用 Hootsuite/Buffer（Web2 工具），不適合幣圈
   - 這是估值跳板：純代理 1-3x → 代理+SaaS 3-5x

2. **亞洲在地化 + 線下活動的組合拳**
   - 多數國際代理商做 PR/KOL 但不做活動
   - 多數活動公司不做數位行銷
   - NodeZ 是「活動 + 內容 + KOL + SaaS」四合一

3. **集團 deal flow 支撐**
   - 鏈嶼/1000X 提供穩定案源
   - 競品需要自己打 BD
   - 但不能過度依賴——需要建立獨立 BD 管線

4. **定價靈活性**
   - 5K-30K USDT 起步，低於歐美競品（8K-50K）
   - 不是「便宜」，是「亞洲價格、國際品質」
   - 績效混合定價（基礎月費 + KPI 獎金）更有吸引力

## 四、NodeZ 官網需要補充的內容（優先級排序）

### P0 — 致命缺失（不補會流失客戶）

| 項目 | 為什麼重要 | 競品怎麼做 | 建議做法 |
|------|-----------|-----------|---------|
| **Logo 牆擴充** | 只有 Sui + Flipster + Haedal，太少 | Coinbound 展示 20+ logos | 加入 1000X、鏈嶼等集團 logo |
| **SaaS 產品展示** | 影響收購估值 180 萬 USDT 差距 | 無競品有，這是藍海 | 加 "Matrix Engine" section + mockup |
| **聯繫表單** | mailto 門檻太高 | Lunar: Calendly、Coinbound: Free Proposal 表單 | 嵌入 Calendly 或 inline 表單 |
| **案例研究** | B 端決策者最看重 | ICODA: 4100% ROI、Coinbound: 699% growth | 至少 1 個 Sui 活動案例（帶數字）|

### P1 — 重要補充（提升專業感）

| 項目 | 建議做法 |
|------|---------|
| **團隊介紹** | CEO + COO 照片 + Title + 一句話背景 |
| **服務流程圖** | 諮詢 → 策略 → 執行 → 月報 → 優化 |
| **定價透明度** | 不需完整定價，但要「方案起步價 + CTA」|
| **客戶推薦語** | 2-3 段帶名字的推薦（可請合作方寫）|

### P2 — 加分項

| 項目 | 建議做法 |
|------|---------|
| **Matrix Engine waitlist** | 收集潛在用戶 email，本身就是收購籌碼 |
| **Dashboard mockup** | 設計一張 SaaS 介面概念圖 |
| **分產業頁面** | Exchanges / Projects / Enterprises 各一頁 |
| **媒體報導 bar** | As Seen In: 放媒體露出的 logo |

## 五、網站架構調整建議

### 現在：
```
首頁 | NodeZ Research (dropdown)
```

### 建議：
```
首頁 | 服務方案 | Matrix Engine | 案例研究 | NodeZ Research | 關於我們
```

**關鍵改變：**
1. **Matrix Engine（SaaS）獨立 nav 位置** — 讓收購方一眼看到這不只是代理商
2. **案例研究獨立頁面** — B 端決策者最常看的頁面
3. **關於我們（含團隊）** — 建立信任
4. **服務方案（含起步價）** — 降低詢價門檻
